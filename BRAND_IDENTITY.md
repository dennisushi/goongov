# GoonGov Brand Identity

## Brand Positioning

**GoonGov** is the transparent governance layer for AI agents. We make the black box transparent, helping teams understand, debug, and trust their AI systems by visualizing execution traces and identifying decision points.

**Tagline:** *"See what your agents see"*

## Brand Personality

- **Playful yet Professional**: The name "GoonGov" suggests a friendly, approachable tool that doesn't take itself too seriously, but delivers serious value
- **Transparent & Trustworthy**: We reveal what's hidden, building trust through visibility
- **Intelligent & Analytical**: We're the detective that finds the smoking gun in complex AI traces
- **Modern & Cutting-Edge**: We're built for the AI era, using the latest visualization and analysis techniques

## Core Values

1. **Transparency First**: No black boxes, no mysteries
2. **Developer-Friendly**: Built by developers, for developers
3. **Actionable Insights**: Not just data, but clear answers
4. **Playful Seriousness**: Serious problems, approachable solutions

## Visual Identity

### Color Palette

**Primary Colors:**
- **Electric Blue** `#0066FF` - Trust, technology, clarity
- **Neon Cyan** `#00D9FF` - Energy, innovation, flow
- **Deep Purple** `#6B46C1` - Intelligence, depth, mystery-solving

**Secondary Colors:**
- **Success Green** `#10B981` - Positive outcomes, AI responses
- **Warning Amber** `#F59E0B` - Tool calls, attention
- **Alert Red** `#EF4444` - Culprits, critical issues
- **Neutral Gray** `#6B7280` - Backgrounds, subtle elements

**Gradients:**
- Primary Gradient: `linear-gradient(135deg, #0066FF 0%, #6B46C1 50%, #00D9FF 100%)`
- Success Gradient: `linear-gradient(135deg, #10B981 0%, #059669 100%)`
- Alert Gradient: `linear-gradient(135deg, #EF4444 0%, #DC2626 100%)`

**Background:**
- **Dark Mode Base**: `#0F172A` (slate-900) - Modern, reduces eye strain
- **Card Background**: `#1E293B` (slate-800) - Depth, hierarchy
- **Elevated Cards**: `#334155` (slate-700) - Interactive elements

### Typography

**Primary Font:** Inter (Google Fonts)
- Clean, modern, highly readable
- Excellent for UI and code
- Weights: 400 (Regular), 500 (Medium), 600 (Semibold), 700 (Bold), 800 (Extrabold)

**Monospace Font:** JetBrains Mono or Fira Code
- For code snippets, tool calls, technical content
- Maintains readability at small sizes

**Typography Scale:**
- H1: 2.5rem (40px) - Brand name, hero text
- H2: 1.5rem (24px) - Section headers
- H3: 1.25rem (20px) - Subsection headers
- Body: 1rem (16px) - Main content
- Small: 0.875rem (14px) - Labels, metadata
- Tiny: 0.75rem (12px) - Badges, tags

### Logo & Brandmark

**Logo Concept:**
- **Wordmark**: "GoonGov" in bold, modern sans-serif
- **Icon**: A stylized eye or magnifying glass with circuit/network lines
- **Treatment**: Can be solid, gradient, or outlined depending on context
- **Placement**: Top-left corner, always visible

**Logo Variations:**
1. **Full Logo**: Wordmark + icon (horizontal)
2. **Icon Only**: For favicons, small spaces
3. **Wordmark Only**: When icon isn't needed

### Visual Language

**Design Principles:**
1. **Glassmorphism**: Subtle frosted glass effects on cards and panels
2. **Neon Accents**: Glowing borders and highlights for interactive elements
3. **Smooth Animations**: Micro-interactions that feel responsive and alive
4. **Generous Whitespace**: Let content breathe, reduce cognitive load
5. **Depth Through Shadows**: Layered shadows create hierarchy
6. **Bold Contrasts**: High contrast for readability and impact

**UI Elements:**
- **Cards**: Rounded corners (12-16px), subtle shadows, glassmorphic backgrounds
- **Buttons**: Gradient backgrounds, hover glow effects, smooth transitions
- **Badges**: Rounded pills, gradient or solid colors, clear hierarchy
- **Inputs**: Clean borders, focus states with glow, modern styling

**Animation Principles:**
- **Purposeful Motion**: Every animation serves a function
- **Smooth Easing**: Use ease-out for natural feel
- **Quick Feedback**: Hover states respond instantly
- **Subtle Delights**: Small surprises that enhance UX

## Tone of Voice

**Writing Style:**
- **Conversational but Precise**: Talk like a smart friend, not a manual
- **Action-Oriented**: Use verbs, be direct
- **Playful When Appropriate**: Don't be afraid of personality
- **Technical When Needed**: Use proper terminology, but explain when helpful

**Examples:**
- ❌ "Execute trace analysis procedure"
- ✅ "Analyze trace" or "Find the culprit"
- ❌ "Error: Invalid input parameters"
- ✅ "Oops! That query needs a bit more detail"
- ❌ "Culprit identification complete"
- ✅ "Found 3 potential culprits"

## Application to Product

### Interface Layout
- **Dark theme** as default (reduces eye strain, feels modern)
- **Left panel**: Execution trace (main focus, larger)
- **Right panel**: Analysis tools and results (secondary, narrower)
- **Top bar**: Minimal, logo + essential controls

### Message Types
- **Human Messages**: Electric Blue (`#0066FF`) - User input, clear and direct
- **AI Messages**: Success Green (`#10B981`) - Agent thinking, positive flow
- **Tool Messages**: Warning Amber (`#F59E0B`) - Actions taken, attention needed
- **Culprits**: Alert Red (`#EF4444`) - Problems identified, urgent attention

### Interactive States
- **Hover**: Subtle glow, slight lift
- **Active/Selected**: Stronger glow, border highlight
- **Loading**: Pulsing animation, gradient shimmer
- **Success**: Green glow, checkmark animation
- **Error**: Red pulse, shake animation

### Data Visualization
- **Trace Flow**: Vertical timeline, clear connections
- **Culprit Highlighting**: Red glow, pulsing animation, clear badges
- **Confidence Scores**: Color-coded (red=high, orange=medium, yellow=low)
- **Tool Calls**: Collapsible, clean formatting, syntax highlighting
